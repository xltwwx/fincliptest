# 案例收集助手 - 微信小程序

这是一个将原有 H5 页面改造成微信小程序的项目，用于收集和管理贷款案例。

## 项目结构

```
miniprogram/
├── app.js                 # 小程序主逻辑（包含全局数据）
├── app.json               # 小程序全局配置
├── project.config.json    # 项目配置文件
├── sitemap.json           # 站点地图配置
├── pages/
│   ├── index/             # 首页（案例列表）
│   │   ├── index.js
│   │   ├── index.wxml
│   │   ├── index.wxss
│   └── detail/            # 详情页（案例编辑）
│       ├── detail.js
│       ├── detail.wxml
│       └── detail.wxss
└── images/                # 图片资源目录
```

## 主要功能

1. **案例列表页**
   - 展示所有案例记录
   - 支持搜索功能（按客户姓名或案例名称）
   - 按日期排序显示
   - 不同类型案例用不同颜色标签区分（正常、纠结、拒贷、逾期）
   - 浮动按钮添加新案例

2. **案例详情页**
   - 查看/编辑案例基本信息（客户号、业务类型）
   - 5 个录音模块：
     - 拟交流案例介绍
     - 客户基本信息与社区化
     - 贷款方案
     - 调查过程
     - 决策争议焦点
   - 录音功能（模拟）：开始/停止录音、计时器、播放、删除
   - 提交确认功能

## 使用方法

### 1. 导入项目

1. 打开微信开发者工具
2. 选择"导入项目"
3. 选择 `miniprogram` 目录
4. 填写 AppID（或使用测试号）
5. 点击"导入"

### 2. 修改 AppID

在 `project.config.json` 中，将 `appid` 字段修改为你自己的小程序 AppID：

```json
{
  "appid": "你的 AppID"
}
```

如果仅用于开发测试，可以使用微信提供的测试号。

### 3. 运行项目

导入后，微信开发者工具会自动编译并预览小程序。

## 与原 H5 版本的区别

1. **移除了 Google 相关代码**：删除了所有 Google AI Studio 相关的依赖和代码
2. **移除了 React 框架**：改用微信小程序原生开发
3. **移除了 Tailwind CSS**：改用微信小程序 WXSS 样式
4. **移除了第三方库**：如 lucide-react、motion 等，使用小程序原生组件和 API
5. **数据存储**：使用小程序全局数据（app.globalData）替代原来的内存存储

## 注意事项

1. **录音功能**：当前为模拟实现，实际使用时需要接入微信小程序的录音 API（`wx.getRecorderManager`）
2. **数据持久化**：当前数据存储在内存中，小程序重启后会丢失。如需持久化，可使用 `wx.setStorage` 或云开发数据库
3. **图标**：当前使用 emoji 和文字图标，可替换为自定义图片图标
